# tz
  个人使用的 github 加速器
  长期维护持更新


# 使用前提
  使用前需要到 【打开应用】 去浏览器端 注册、登录，并支付少量服务器成本费用
  
  为了长期稳定使用，以及，创造健康良好的网络环境 ，本人坚持如下原则：
  1. 仅用作浏览知识，提升自我（毕竟很多技术学习资料网内真的很难找啊~）
  2. 谢绝用作违法勾当，一旦发现，本人将取消该用户的使用资格
  3. 禁止未成年人浏览不健康的网页
  4. 禁止成年人散播非健康网页和内容
